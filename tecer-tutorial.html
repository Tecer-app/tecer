Tutorial breve

Aprende a usar o Tecer em minutos com estes tutoriais rápidos.

Criação de mapa

Aprende a criar o teu primeiro mapa de argumentos — desde a conclusão principal até às premissas e objeções que a sustentam.

  1. 1Abre o Tecer e inicia uma tela em branco
  2. 2Cria o nó central com a tua conclusão principal
  3. 3Adiciona nós de premissas e liga-os à conclusão
  4. 4Marca suposições implícitas e possíveis objeções

Atalhos e ação

Domina os atalhos de teclado e as ações rápidas que tornam o Tecer mais fluido e eficiente no teu fluxo de trabalho.

  1. 1Duplo clique num nó para editar o seu texto
  2. 2Arrasta do bordo de um nó para criar uma ligação
  3. 3Usa as teclas direcionais para navegar no mapa
  4. 4Pressiona Delete para remover o nó selecionado

Gestão da sessão

Aprende a guardar, exportar e reimportar os teus mapas. Os teus dados ficam sempre no teu browser — sem servidores ou contas.

  1. 1Os mapas são guardados automaticamente no browser
  2. 2Exporta em SVG para partilhar ou imprimir com qualidade
  3. 3Exporta em JSON para fazer uma cópia de segurança
  4. 4Importa um ficheiro JSON para retomar o trabalho
